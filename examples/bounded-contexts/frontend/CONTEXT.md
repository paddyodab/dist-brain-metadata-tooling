# Frontend context

## Glossary

### OrderView
The page-level React component that renders an order summary for a customer.

### LineItemRow
A sub-component that displays one LineItem inside an OrderView.

### CartBadge
A small indicator showing the number of items currently in the cart.

### CheckoutFlow
The multi-step frontend wizard that collects payment and shipping details before submitting to the backend.

### Renders
The action of producing JSX output from a React component.
